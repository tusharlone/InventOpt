Welcome to InventOpt's documentation!
===================================

**InventOpt** is a Python Library for modelling, simulation, design exploration and optimization of Supply Chains and Inventory Problems. InventOpt packs tools to model and simulate a supply chain network, set up the experimental environment, and explore its design space visually. It is a modular library comprised of various optimization methods, including simulation-based, to analyze and optimize supply chain and inventory systems.

Check out the :doc:`getting started` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
